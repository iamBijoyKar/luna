<h1>Luna</h1>
<p>
    Luna is an ollama wrapper with extra features. Written in JavaScript.
</p>
</div>

## Features 🌟
- [x] **Run on terminal easlily**
- [x] **Generated text will be saved in your clipboard**
- [ ] **Select model from ollama**

## Prerequisites 📋
- Ollama 
- Node.js
